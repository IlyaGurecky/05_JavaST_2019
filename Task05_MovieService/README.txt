About:
My project: movie service.
Roles in the site: admin, films editor, user.

User capabilities: view films information, register an account,
add film to "See later" list, change profile information and password,
find film by name, view films by category, watch film(after pressing the
button "Watch a movie", the movie is added to the list "Watched". If the movie
has already been watched, then the viewing date changes).

Admin capabilities: create editor account, admin account, user account,
add film, delete films and users.

Editor capabilities: change film information, image.
