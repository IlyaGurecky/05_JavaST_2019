This program lets to user calculate square and perimeter of quadrangle.

Input data in file: 4 2D dots (8 numbers). The points must be located clockwise
or counterclockwise.

To calculate parameters program uses algorithms based on geometrical
mathematical formulas(Gauss Square Formula for example).

Code written with plugins "CheckStyle" and "SonarLint".

Using technologies: TestNG, Log4j2, Java stream API.
