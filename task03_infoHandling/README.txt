This app allows you to create text tree.
You can sort the text in various ways(Text by paragraphs,
sentences by words length, text by symbol occurrences in lexemes).
Also, you can calculate byte expressions(At first create polish notation).

File data: correct text, in which all sentences begin with a uppercase letter,
paragraphs start after 4 spaces. All sentences ends with [.?!...].

Patterns: Composite, Chain of responsibility, Interpreter.

Using technologies: TestNG, log4j2, Java StreamAPI.
